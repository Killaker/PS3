このパッケージは、PlayStation(R)3用タイトルを納品していただくための
マスターディスクやマスターパッケージを作成したり検証したりするための
ソフトウェアで構成されるツールパッケージです。

このパッケージには以下のソフトウェアが含まれています。
- Disc Image Generator for PlayStation(R)3
- System File Utility for PlayStation(R)3
- Disc Image Checker for PlayStation(R)3
- Trophy Pack File Utility for PlayStation(R)3
- Disc Image Generator for PlayStation(R)3 Command Line version

注意事項
----------------------------------------------------------------------
- マスターディスクを納品する際には、Version 3.00以降のGeneratorまたは
  Checkerを用いてマスターディスクの検証を行い、エラーが無いことを
  必ず確認してください。
  CheckerとGeneratorはバージョンが等しければ同等の検証機能を持ちます。
  マスターディスクの検証方法につきましては、ユーザガイドを
  参照してください。

- SDK350かそれ以降の開発環境を用いてマスターディスクやパッケージを
  作成する際には、Version 3.00かそれ以降のGeneratorを使用してください。

- SDK341かそれ以降の開発環境を用いてマスターディスクやパッケージを
  作成する際には、Version 2.89かそれ以降のGeneratorを使用してください。

- SDK330かそれ以降の開発環境を用いてマスターディスクやパッケージを
  作成する際には、Version 2.85かそれ以降のGeneratorを使用してください。

- ゲームマスターディスク作成規定マニュアルにつきましては、
  ドキュメントパッケージにて提供しています。

- 以下のドキュメントは、Generator Toolsユーザガイドに統合されました。
  ・Disc Image Generator for PlayStation(R)3 ユーザマニュアル
  ・Trophy Pack File Utility for PlayStation(R)3 ユーザガイド

Version 3.00 の新規機能と変更点
----------------------------------------------------------------------
<新規機能>
[ps3gen.exe, ps3sys.exe]
- 以下のパラメータを設定する機能が追加されました。
  - PlayStation(R)Moveモーションコントローラの地磁気センサの
    キャリブレーションが行われていない場合に、ユーザへの通知を行うかどうか。
  - 3D立体視に関する注意喚起文を、ゲーム起動時に表示するかどうか。

<変更点>
[全般]
- Generator/Checkerの検証機能の仕様が変更されました。
  - SDK350向けのライブラリの検証に対応しました。
  - エクストラコンテンツ用静止画アイコンのサイズ制限(最大300kB)が
    撤廃されました。

<不具合修正>
[全般]
- 以下の不具合を修正しました。
  - 圧縮されたSPRXファイルの検証に失敗する。

---------------------------------------------------------------------
既知の不具合・制限事項
---------------------------------------------------------------------
- インストールパッケージ、コンテンツデータあるいはディスク起動ゲームに
  自動で適用されるパッチパッケージを含むマスターディスクを提出する予定
  の場合は、事前にSCEに確認を取ってください。

- HDD起動ゲーム用のシステムファイルのパラメータとHDD起動ゲームの累積パッチ
  用のパラメータは区別がつかないため、HDD起動ゲームの累積パッチ用のシステム
  ファイルをps3sys.exeで開いた際にはPatchチェックボックスの状態が
  正しく反映されません。そのようなシステムファイルを編集する際には、
  最初にPatchチェックボックスをオンにするようにしてください。

- Enable BGM チェックボックスに関する注意事項につきましては、
  以下の技術情報を参照してください。
  (技術情報ID:200807-27  https://ps3.scedev.net/technotes/view/779)

- Windows VistaでBD Emulator機能を用いる場合には、Generatorを管理者
  として実行する必要があります。

- Generatorでメディアに書き込みを行う場合は、スクリーンセーバ等の
  Generator以外のアプリケーションを動作させないようにしてください。

- DVDメディアはサポート外となりましたためマニュアルには記載されて
  おりませんが、DVD 4.7GB Single Layerのディスクイメージをメディアに
  書き込む場合は DVD-R SLまたはDVD+R SLを、DVD 8.3GB Dual Layerの
  ディスクイメージをメディアに書き込む場合はDVD+R DLをご使用下さい。
  Reference ToolはDVD-RW/DVD+RW等をゲームマスターディスクとして
  認識いたしません。

- ディスク種類に"DVD 8.3GB Dual Layer OTP"を選択した場合、レイアウト
  タブでLayer1に配置されるファイルが青色(Layer0に配置されるファイルの
  表示色)で表示されることがあります。

- ゲーム起動用のコンテンツ情報ファイル(アイコン等)の表示機能で表示可能な
  内容は PNGファイルのみです。

- Generatorのプロジェクトにファイルを追加した後に、エクスプローラや
  他のアプリケーションなどでそのファイルを編集したり、移動してから
  ディスクイメージを作成すると、Generatorに追加した時のファイルサイズで
  イメージを作成します。
  そのため、ディスクイメージの作成に失敗したり、ディスクイメージ上の
  ファイルが意図しないサイズになることがあります。

- 書き込み機能に関して動作確認済みのドライブ・メディアは以下の通りです。
  - ドライブ
    - パイオニア BDR-101A (BD 50GBには非対応)
    - パイオニア BDR-202 (BD 50GBには非対応)
    - パナソニック SW-5582
    - パナソニック SW-5584
    - パナソニック LF-MB121
    - パナソニック LF-PB271
    - 日立LG GGW-H20N
    - 上記と同じプロダクトIDを持つOEMモデル
  - メディア
    - BD-RE SL(25GB)
      - ソニー 全ての2倍速メディア
      - TDK BDD-RE25S
      - マクセル BDRE25P.1P
    - BD-R SL(25GB)
      - ソニー 全ての2倍速/4倍速/6倍速メディア
      - TDK BDD-R25S, BDD-R25A, BRD25PWA10S
      - パナソニック LM-BR25LD
    - BD-RE DL(50GB)
      - ソニー 全ての2倍速メディア
      - パナソニック LM-BE50D
    - BD-R DL(50GB)
      - ソニー 全ての2倍速/4倍速/6倍速メディア
      - パナソニック LM-BR50D
      - TDK BDD-R50A
  - BD-R LTHメディアに対する書き込みの動作確認は行なっておりません。
    また、BD-R LTHメディアをマスター提出用に使用することはできません。
  - BD-R 4倍速メディアへの書き込みには、BD-R Ver1.2に対応したドライブが必要です。
  - BD-R 6倍速メディアへの書き込みには、BD-R Ver1.3に対応したドライブが必要です。
  - 初期化ずみBD-Rメディアを使用しないでください。

- 以下のドライブは、マスターディスクの作成に制限があることが
  確認されています。
  - PLDS(Philips/LITE-ON) DH-4B1S (rev.7D14)
    - ドライブの仕様あるいは不具合のため、BD-Rメディアへイメージを
      書き込む際にドライブは自動的に交代領域を作成します。
      このためこのドライブでは、Generatorでも市販のライティング
      ソフトでもPlayStation(R)3用のBD-Rマスターディスクを作成する
      ことができません。
      BD-REメディアを使用するのであれば、このドライブで
      問題なくマスターディスクを作成することができます。
